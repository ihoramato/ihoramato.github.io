生化 脂质代谢 题目

发表于2024-12-04|更新于2024-12-06|生物化学
|总字数:1.2k|阅读时长:3分钟

名词解释

ACAT

酯酰辅酶A胆固醇酯酰转移酶。分布于细胞内质网，将脂酰基转移至胆固醇的羟基上，使胆固醇储存于胞液中。

ACP

酰基载体蛋白。是脂肪酸合成过程中脂酰基的载体，脂酸合成各部反应均在ACP的辅基上进行。

乳糜微粒

脂蛋白的形式。由小肠粘膜细胞的甘油三酯和载脂蛋白，磷脂等合成。是外源性甘油三酯和胆固醇的主要运输形式。

必需脂肪酸

人体自身不能合成的脂酸。包括亚油酸、亚麻酸、花生四烯酸

脂肪动员

脂肪细胞中的甘油三酯，逐步水解为甘油和脂肪酸，血液运输到其他组织利用

LCAT

卵磷脂胆固醇酯酰转移酶。血浆中催化胆固醇酯化的酶。将HDL表面脂酰基转移至胆固醇羟基生成溶血卵磷脂和胆固醇酯。促进胆固醇逆向转运。

脂蛋白脂酶

位于肌肉、脂肪组织等的毛细血管内皮细胞表面，水解CM和VLDL的甘油三酯的酶。

血脂

血浆中所含的总脂类，包括甘油三酯，磷脂，胆固醇（酯），游离脂酸等

脂酸的活化

脂酸必须先转变为酯酰辅酶A才能继续分解

CDP-甘油二酯途径

磷脂酰肌醇，磷脂酰丝氨酸和心磷脂的主要合成方式。先结合CTP，再和取代基结合

β氧化

酯酰COA经肉碱穿梭运行至线粒体基质，从B碳原子开始反复脱氢、加水、再脱氢、硫解的过程，终产物为乙酰COA和还原性辅酶

解答题

计算1mol软脂酸的ATP数

C16-8乙酰COA、7FADH2、7NADH
71.5+72.5+8*10-2=106

酮体生成和利用的流程
  1. 乙酰辅酶A-乙酰乙酰辅酶A-HMGCOA-乙酰乙酸-β羟丁酸、丙酮（关键酶是HMGCOA合成酶）
  2. 肝内不能利用酮体。先转化为乙酰辅酶A，再经过三羧酸循环彻底氧化。
酮体的生理意义
  1. 易运输的形式
  2. 减少糖摄取，维持血糖稳定
  3. 饥饿时的能量供应
胆固醇合成的原料、部位、限速酶、中间产物、去路
  1. 原料：乙酰COA,NADPH,H+,ATP
  2. 部位，全身各组织，肝最多
  3. 限速酶：HMGCOA还原酶
  4. 中间产物：-HMGCOA-甲羟戊酸-鲨烯-
  5. 去路：
    1. 细胞膜
    2. 胆汁酸
    3. 性激素、肾上腺皮质激素
    4. 维生素D3
    5. 胆固醇酯储存
脂肪酸分解的过程
  1. 脂肪动员
    1. 脂肪-甘油二酯+脂酸（ATGL）
    2. 甘油二酯-甘油一酯+脂酸（HSL）
    3. 甘油一酯-甘油+脂酸
    4. 抑制——胰岛素、前列腺素；激活：甲状腺激素、胰高血糖素
  2. 利用甘油
    1. 2分子甘油糖异生：消耗2ATP
    2. 1分子甘油完全分解：18.5/16.5ATP
  3. β氧化
脂肪酸合成的过程
  1. 甘油来源：磷酸二羟丙酮（脂肪和骨骼肌只能）、脂肪动员的甘油
  2. 脂肪酸来源：乙酰COA（柠檬酸-丙酮酸循环）、NADPH\ATP、乙酰COA羧化酶
  3. 合成软脂酸（胞浆、ACP载体）
  4. 合成后延长碳链（内质网或线粒体，COA载体）
为什么胆碱缺乏会导致脂肪肝？

卵磷脂合成减少，VLDL合成障碍

脂质的生理功能
  1. 储能功能
  2. 脂酸是活性物质
  3. 类脂参与生物膜信息
  4. 转化为其他生理活性物质
游离脂酸在干细胞的主要代谢去路
  1. 三酰甘油和磷脂
  2. 生成乙酰COA
血浆脂蛋白的结构
  1. 疏水的三酰甘油（CM\VLDL）和胆固醇酯（LDL\HDL）位于内核
  2. 载脂蛋白、磷脂和游离胆固醇在表面，球状
载脂蛋白的作用
  1. 脂类载体（但游离脂肪酸靠清蛋白转运）
  2. 脂类交换的连接蛋白
  3. 脂类受体的识别标志（A1-HDL,B100\E-LDL）
  4. 脂类代谢酶的调节剂（A1-LCAT,C2-脂蛋白脂肪酶）
合成胆固醇的影响因素
  1. 饥饿？
  2. 胆固醇含量
  3. 激素
四种血浆脂蛋白的化学组成、合成部位、功能
试述LDL受体代谢途径
  1. LDL和受体结合，内吞，与溶酶体结合
  2. APOB100等水解为氨基酸，胆固醇酯被水解为游离胆固醇
    1. 被细胞膜摄取
    2. 抑制自身胆固醇合成
    3. 抑制LDL受体蛋白合成
    4. 激活ACAT
乙酰COA的来源和去路

来源：

  1. 糖酵解-丙酮酸
  2. 脂肪分解
  3. 蛋白质-α酮酸

去路：

  1. 三羧酸循环
  2. 合成脂肪酸和胆固醇
  3. 合成酮体
