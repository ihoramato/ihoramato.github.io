生化 糖代谢 题目

发表于2024-12-01|更新于2024-12-04|生物化学
|总字数:1.4k|阅读时长:4分钟

名词解释

糖酵解

1分子葡萄糖在胞质中生成2分子丙酮酸，是无氧氧化和有氧氧化的共同起点，

有氧氧化

利用氧将葡萄糖彻底分解为CO2和H2O的过程，是体内糖分解供能的主要方式。

TCA循环

即三羧酸循环、柠檬酸循环

巴斯德效应

有氧氧化抑制生醇发酵的效应

糖异生

非糖化合物转化为葡萄糖或糖原的过程

磷酸戊糖途径

从G6P开始形成旁路，通过氧化、基团转移两个步骤生成M6P和3-磷酸甘油醛，重新进入糖酵解。

解答题

糖无氧氧化的意义
  1. 快速提供能量，尤其是肌肉
  2. 成熟红细胞没有线粒体
  3. 白细胞，神经细胞，骨髓细胞代谢旺盛，即使有氧也要无氧氧化
有氧氧化包括几个阶段？关键酶及催化的反应是什么？
  1. 糖酵解：己糖激酶、磷酸果糖激酶-1、丙酮酸激酶
  2. 丙酮酸生成乙酰COA：丙酮酸脱氢酶复合体
  3. 三羧酸循环：柠檬酸合酶、异柠檬酸脱氢酶、a-酮戊二酸脱氢酶
简述柠檬酸循环的特点及意义

特点：

  1. 4次脱氢（3次NADH，琥珀酸-延胡索酸FADH2），2次脱羧，一次水平磷酸化
  2. 3个关键酶
  3. 草酰乙酸由丙酮酸直接羧化或苹果酸脱氢产生

意义：

  1. 获取能量的主要方式
  2. 三大营养物质的共同通路
  3. 三大营养物质联系的枢纽
磷酸戊糖途径包括几个阶段？关键酶和代谢的反应是什么？
  1. 氧化反应——磷酸核糖、NADPH、CO2
  2. 基团转移——6-磷酸果糖和3-磷酸甘油醛
  3. 关键酶：G6P脱氢酶
糖异生的关键酶是什么？
  1. 丙酮酸羧化酶
  2. 磷酸烯醇式丙酮酸羧激酶
  3. 果糖二磷酸酶-1
  4. 葡糖-6-磷酸酶
葡糖-6-磷酸的作用
  1. 继续糖代谢
  2. 糖异生
  3. 变成G1P进入糖原合成或糖醛酸途径
  4. 进入磷酸戊糖途径
乳酸循环及其生理意义
  1. 肌细胞-乳酸-血液-肝-葡萄糖-血液-肌肉，又称cori循环
  2. 回收乳酸的能量/避免酸中毒
血糖的来源和去路

来源：

  1. 食物消化吸收
  2. 肝糖原分解
  3. 糖异生

去路：

  1. 氧化分解
  2. 合成肝糖原或肌糖原
  3. 变为其他糖
  4. 合成脂肪或氨基酸
糖异生的生理意义
  1. 空腹时维持血糖
  2. 肝补充或恢复糖原
  3. 肾糖异生维持酸碱平衡
磷酸戊糖途径的生理意义
  1. 提供5-磷酸核糖
  2. 提供NADPH
肝糖原合成的直接途径和间接途径
  1. 直接：葡萄糖-UDPG
  2. 间接：葡萄糖-三碳化合物-肝脏-糖异生
草酰乙酸在糖代谢中的作用
  1. 三羧酸循环
  2. 糖异生的原料
  3. 丙酮酸、氨基酸等糖异生的中间产物
B族维生素在糖代谢中的作用
  1. 生成1.3-二磷酸甘油酸需要VIT PP
  2. 丙酮酸和a-酮戊二酸需要TPP、硫辛酸、FAD、Coa、NAD+
  3. 异柠檬酸和苹果酸需要PP
  4. 琥珀酸需要B2
肝在糖代谢中的作用
  1. 糖原合成和分解
  2. 糖异生
  3. 果糖等变为葡萄糖
  4. 糖醛酸和磷酸戊糖代谢
为什么高碳水饮食会使葡糖6磷酸脱氢酶的活性升高？

生成脂肪酸，NADPH消耗

糖酵解和糖异生的调节

别构效应剂：

  1. ATP和柠檬酸激活果糖二磷酸酶-1，抑制磷酸果糖激酶-1
  2. AMP和果糖26-二磷酸激活磷酸果糖激酶，抑制果糖二磷酸酶
  3. ATP激活丙酮酸羧化酶，抑制丙酮酸激酶
  4. 乙酰COA激活丙酮酸羧化酶，抑制丙酮酸脱氢酶复合体

激素：

  1. 胰岛素：关键酶增强\抑制
  2. 胰高血糖素：抑制果糖二磷酸生成\丙酮酸激酶，诱导磷酸烯醇式丙酮酸羧激酶
低血糖的原因
  1. 胰性
  2. 肝性
  3. 内分泌异常
  4. 肿瘤
  5. 饥饿或不嫩进食
乳酸变为葡萄糖的过程
  1. LDH催化生成丙酮酸
  2. 草酰乙酸-天冬氨酸（AST）-出线粒体-草酰乙酸
  3. 糖异生
丙酮酸可进入哪些代谢途径？
  1. 缺氧时——乳酸
  2. 供氧时——三羧酸循环
  3. 糖异生
  4. 草酰乙酸-柠檬酸-三羧酸循环
  5. 草酰乙酸-柠檬酸-乙酰coa-脂质合成
  6. 生成氨基酸
乳酸氧化供能的反应
  1. LDH催化——丙酮酸、NADH
  2. 丙酮酸脱氢酶——乙酰coa、NADH、CO2
  3. 三羧酸循环——3NADH,1NADH2,2CO2
列表比较糖酵解和有氧氧化
  1. 反应条件
  2. 反应场所
  3. 关键酶
  4. 产物
  5. 能量
  6. 生理意义
机体如何调节糖原的合成和分解
  1. 磷酸化酶：磷酸化酶b（去磷酸化）无活性，磷酸化酶a（磷酸化）有活性，磷蛋白磷酸酶催化。
  2. 糖原合酶：糖原合酶a（去磷酸化）有活性，糖原合酶b（磷酸化）无活性、胰高血糖素和肾上腺素激活腺苷酸环化酶-cAMP-蛋白激酶A-糖原合酶磷酸化
肾上腺素对血糖的调节
  1. G蛋白-cAMP-蛋白激酶A
  2. 磷酸化酶激活，糖原合酶抑制，丙酮酸激酶抑制，果糖二磷酸酶-1激活
解释为什么减肥要减少糖摄入
  1. 葡萄糖-丙酮酸
  2. 丙酮酸-乙酰COA-（柠檬酸丙酮酸循环）—乙酰COA-脂肪酸
  3. 磷酸二羟丙酮-α-磷酸甘油-甘油三酯
