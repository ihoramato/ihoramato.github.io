生理学 血液循环 题目

发表于2024-12-03|更新于2024-12-08|生理学
|总字数:1.3k|阅读时长:3分钟

名词解释

心动周期

从一次心跳的起始到下一次心跳的起始，心血管系统的经历的过程

每搏输出量

一次心搏一侧心室的射血

射血分数

每搏输出量占心室舒张末期容积量（前负荷）的百分比

心指数

心脏的输出量除以体表面积

心力贮备

心输出量随机体代谢的需要而增加的能力。包括搏出量储备（收缩期+舒张期）和心率储备

心音

由于心肌收缩
心脏瓣膜关闭和血液撞击心室壁、动脉壁引起的声音。分为第一心音和第二心音。

自律性

心肌在无外刺激存在的条件下能自动产生节律性兴奋的能力。

窦性节律

窦房结发出激动所形成的心律，约每分钟60-100次

异位心律

由窦房结意外的其他起搏点控制心脏形成的心率

房室延搁

兴奋在房室交界处缓慢传播耗时较长的现象称为房室延搁。确保心房和心室收缩的先后顺序。

期前收缩

指异位起搏点发出的过早冲动引起的搏动

循环系统平均充盈压

当心脏停止跳动时，循环系统内各处的压力都相等，这个压力就是循环系统平均充盈压。反应循环血量和血容量的比

平均动脉压

一个心动周期中每个时刻动脉血压的平均值

中心静脉压

右心房及上下腔静脉胸腔段的压力

血流量

单位时间内流经血管某一截面的血量

心肺感受器

位于心房、心室和大血管壁上。是容量感受器

心房钠尿肽

心房肌细胞合成并释放的肽类激素。具有强烈的舒张血管，降血压，利尿作用

血脑屏障

由血管内皮细胞和星形胶质细胞组成。阻止有害物质由血液进入脑组织

心室功能曲线

以舒张末期容积（充盈压）为横轴，搏功为纵轴的曲线。反映前负荷和搏功的关系。

解答题

心输出量的正常值

安静状态下约5L，运动时可达25-30L

第一心音和第二心音的形成的原因
  1. 第一心音：二尖瓣和三尖瓣关闭。代表心室收缩开始
  2. 第二心音：动脉瓣关闭。代表心室舒张开始
什么是等长调节
什么是异长调节
心室肌细胞动作电位的产生机制
  1. 静息电位-90mv，阈电位-40mv
  2. 0期：INa开放
  3. 1期：Ito（K外流）
  4. 2期：Ica-L；IK1；IK
  5. 3期：IK1为主
  6. 4期：没有自动去极化
心肌细胞的生理特性
  1. 兴奋性（膜电位，阈电位，通道状态）
  2. 传导性（去极化速度，直径，临近细胞兴奋性）
  3. 收缩性（全或无、无明显降支、单收缩、依赖细胞外Ca）
  4. 自律性
影响心肌细胞兴奋性的因素
  1. 静息电位
  2. 阈电位
  3. 引起0期去极化的离子通道的活性
影响自动去极化的因素
  1. 最大复极点位
  2. 阈电位
  3. 4期自动去极化速度
影响传导性的因素
  1. 结构因素（直径，连接方式）
  2. 膜电位
  3. 临近未兴奋细胞的兴奋性
心肌收缩的特点
  1. 全或无
  2. 不发生强直收缩
  3. 对胞外Ca的高依赖性
动脉血压的正常值

收缩压100-120mmHg
舒张压30-40mmHg
脉压60-80mmHg
平均动脉压100mmHg

微循环通路有哪些？
  1. 迂回通路
  2. 直捷通路
  3. 动静脉短路
中心静脉压的正常值是多少？取决于什么？

4-12cm H2O，取决于心脏射血能力和回心血量的关系

影响静脉回流的因素有哪些?
  1. 体循环平均充盈压
  2. 心脏收缩力
  3. 体位
  4. 骨骼肌的挤压作用
  5. 呼吸运动
调节心血管的局部体液因素有？
  1. 激肽类物质
  2. 组胺
  3. 组织代谢产物（腺苷，CO2，乳酸等）
冠脉循环的特点
  1. 毛细血管网极为丰富
  2. 途径短，血流快，血流量大
  3. 摄氧率高，耗氧量大
比较心室肌细胞、窦房结、浦肯野纤维
影响动脉血压的因素
  1. 每搏输出量
  2. 心率
  3. 外周阻力
  4. 大动脉弹性
  5. 循环血量和系统容量的比例
颈动脉窦、主动脉弓的压力感受性反射的过程

感受器-窦神经主动脉神经-延髓中枢-孤束核-迷走神经-心率减慢，搏出量减少，血管舒张

  1. 有效胶体渗透压
  2. 有效流体静压
  3. 毛细血管的通透性
  4. 淋巴回流

肺循环的生理特点

  2. 血容量大、变化大
  3. 肺毛细血管有效滤过压低，保持干燥
